Otra diferencia JavaScript ha sido un lenguaje interpretado, y Java compilado. Por parte de las semejanzas, Ambos son lenguajes de programación populares utilizados en el desarrollo de aplicaciones web, Ambos lenguajes pueden interactuar con el DOM (Document Object Model) para crear aplicaciones web interactivas.

  5. ¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

    Puedes encontrar muchas razónes para no hacerlo, pero las mas importantes son: Confusión y legibilidad: Si una variable global se utiliza dentro de una función, puede ser difícil entender su origen y su propósito; Y seguridad, en cuánto a las variables es mas seguro siempre tener variables locales.
